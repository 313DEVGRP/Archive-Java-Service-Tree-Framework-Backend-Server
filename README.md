### jsTree Service Framework란: ###

MVC패턴을 구현한 다양한 프레임워크를 대상으로 ( EgovFramework 기반한 ) 서비스레이어의 도메인 객체를 Tree Base Object로 간주하여 서비스 - DAO - DTO - DB를 정형화한 기반 구현체를 제공하는 프레임워크 입니다.

특장점 :

- 중앙집중형 아키텍쳐
- 확장(extend)을 염두해 둔 객체지향 개발 방식
- 안정성
- Speeeeeeeeeeed~!
